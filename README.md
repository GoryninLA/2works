# gRPC Project (Ubuntu 20.04.6 LTS)

##  Описание проекта
Проект реализует клиент-серверное взаимодействие с использованием технологии gRPC на языке Python.  
Сервер и клиент обмениваются данными через протокол Protocol Buffers.

##  Цель проекта
Изучение и практическая реализация:
- удалённых вызовов процедур (RPC)
- работы с .proto файлами
- генерации кода через grpcio-tools
- клиент-серверной архитектуры

##  Технологический стек
- OS: Ubuntu 20.04.6 LTS
- Python 3.8+
- venv
- grpcio
- grpcio-tools
- Protocol Buffers
